## App overview

- Angular 4 ( 4.x )
- ExpressJS ( 4.x - with compression )
- Webpack ( angular-cli )

## Install / Run

```bash
git clone https://github.com/panadrota/angular4-twitter.git
cd angular4-twitter

# Install dependencies
npm install

# start server
npm run start

# Client url: http://localhost:4200
# Application ( epxress ) API: http://localhost:4300
```
